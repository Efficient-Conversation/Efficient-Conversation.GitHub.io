# Efficient Conversation's Website

## Built With

* [Jekyll](https://jekyllrb.com/) - Static Site Generator

## Instructions

To run you will need to install [Ruby+Devkit](https://rubyinstaller.org/downloads/) and then after that is intalled open Powershell and do `gem install bundler jekyll`. 
Once that completes you will need to go into the cloned directory and in Powershell run `bundle install`, after that youre ready to run!

To start the webserver just run `bundle exec jekyll serve`.
